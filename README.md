## Getting Started

### Prerequisites   (#optional)
- Python 3.x
- pip (Python package manager)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/saikumar-pujari/System-design-LLD-.git
   cd System-design-LLD-
   ```

2. **Activate the virtual environment:**        (#no need to create virtual environment)
   ```bash
   # On Windows (CMD)
   env\Scripts\activate.bat
   
   # On Windows (PowerShell)
   env\Scripts\Activate.ps1
   
   # On macOS/Linux
   source env/bin/activate
   ```

3. **Install dependencies:**        (#if any dependencies are added in future)
   ```bash
   pip install -r requirements.txt
   ```
